## Quick Start

```bash
# Install
npm install remembrance-oracle-toolkit

# Initialize (loads patterns, installs hooks, syncs library)
npx remembrance-oracle-toolkit init

# Search for proven patterns
oracle search "rate limiter"

# Smart retrieval
oracle resolve --description "retry with backoff" --language javascript
# → { decision: 'PULL', pattern: { name: 'retry-with-backoff', coherency: 0.96 } }

# Score your code
oracle reflect myfile.js

# Start MCP server (for AI agents)
oracle mcp
```
