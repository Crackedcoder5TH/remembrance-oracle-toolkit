# Remembrance Ecosystem — Acceptable Use Policy

**Effective Date:** April 10, 2026

This Acceptable Use Policy ("AUP") defines prohibited activities when using the Remembrance Services.

---

## Prohibited Activities

You may NOT use the Services to:

### 1. Substrate Extraction
- Bulk-download, scrape, or systematically extract patterns from the Substrate Data
- Build a competing pattern database using data obtained through our API
- Use automated tools to enumerate or export substrate contents
- Reverse-engineer waveform signatures to reconstruct source code

### 2. Security Violations
- Attempt to bypass authentication, rate limiting, or access controls
- Share, publish, or embed API keys in public repositories or client-side code
- Probe, scan, or test the vulnerability of the Services
- Interfere with or disrupt the Services or associated infrastructure

### 3. Harmful Code
- Generate, submit, or register malicious code (malware, ransomware, exploits)
- Attempt to circumvent the Covenant safety filter
- Use the Services to develop tools intended for unauthorized access to systems
- Submit code designed to attack, disrupt, or damage third-party systems

### 4. Abuse
- Exceed rate limits through automated retry loops or distributed requests
- Create multiple free accounts to circumvent tier limitations
- Resell, sublicense, or redistribute paid tier access
- Misrepresent your identity, affiliation, or authorization level

### 5. Content Violations
- Register patterns that contain hardcoded secrets, credentials, or PII
- Submit code that violates intellectual property rights of third parties
- Use the Services to process data subject to legal restrictions (ITAR, EAR) without appropriate authorization

---

## Enforcement

Violations of this AUP may result in:
1. **Warning** — First-time minor violations
2. **Rate limiting** — Reduced API access for 24-72 hours
3. **Suspension** — Temporary account lockout with 30-day data export window
4. **Termination** — Permanent ban for serious or repeated violations

We may report illegal activities to law enforcement authorities.

---

If you become aware of a violation of this AUP, please report it:
- GitHub Issues: https://github.com/Crackedcoder5TH/remembrance-oracle-toolkit/issues
