Summary

Key points at a glance

The full legal documents are the authoritative source. These highlights are provided for convenience.

✓

We do not store your source code unless you explicitly register it as a pattern via oracle register or oracle submit. Code submitted for scoring is processed in memory and discarded.

✓

We do not read your private repositories. The oracle runs locally. Cloud sync only transfers pattern metadata (name, tags, coherency score, waveform) — not raw source code.

✓

We do not train AI models on your code. Substrate waveforms are one-way compressed signatures. Your code cannot be reconstructed from them.

✓

Open source code, proprietary substrate. All Remembrance source code is MIT-licensed. The substrate data (pattern libraries, waveform databases, domain substrates) is proprietary and licensed separately.

✓

You own your patterns. Patterns you create from your code belong to you. Community-shared patterns are licensed under MIT. We do not claim ownership of user-created patterns.

✓

No data sharing for advertising. We do not share your data with third parties for advertising purposes. Usage metrics are anonymized and used only to improve the service.

✓

Substrate extraction is prohibited. You may not bulk-download, scrape, or systematically extract patterns from the proprietary substrate data. Building a competing pattern database from our API is not permitted.

✓

Self-hosted is always an option. The entire open-source toolkit runs locally with zero external dependencies. Your patterns stay on your machine. No cloud account required for the Community tier.

✓

Payments are processed by Stripe. We never store credit card numbers. Payment processing is handled entirely by Stripe with PCI-DSS compliance.

All Remembrance source code is open source under the MIT License. This includes the Oracle Toolkit library, CLI tools, VS Code extension, Reflector, Swarm, Dialer, and Dashboard. Fork it, modify it, ship it. Attribution appreciated but not required.

The proprietary pattern libraries, waveform databases, and domain substrate files are licensed separately and available through paid plans. These power the extended pattern matching and quality scoring capabilities beyond the 302 built-in patterns.
